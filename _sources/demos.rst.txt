Demonstrations
==============

Here we provide some demonstrations of OpenAirClim. 
These demonstrations were tested with oac v0.11.1.

.. note::
    
    Before running the demonstrations by yourself, make sure that the file paths,
    defined in the code blocks as well as those paths defined in the configuration files,
    have been changed according to your settings.
