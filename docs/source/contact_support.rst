Contact and Support
===================


Contact
-------

Please use the `discussions tab <https://github.com/dlr-pa/oac/discussions>`_ on GitHub for general questions or requests for help with your specific project.
GitHub issues are reserved for bug reporting and feature requests. For everything else, send a message to openairclim@dlr.de 


Support
-------

OpenAirClim is a joint, open-source effort. We greatly value contributions of any kind!
Before submitting issues and pull requests, please familiarise yourself with our `contributing guidelines <https://github.com/dlr-pa/oac/blob/main/CONTRIBUTING.md>`_.


Funding
-------

We are grateful for the financial support from our industrial partner Airbus.
